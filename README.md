# Tile Creation Platformer (Mario Maker Style Game)

## Overview
This project is a tile-based platformer game developed in Java as part of a software engineering course. It was completed by a team of three students. The game allows players to design custom levels by selecting and placing different tile types, then spawn a character and attempt to reach the end while navigating through the created level.

The project is inspired by sandbox-style level editors such as Mario Maker, focusing on both level creation and gameplay mechanics.

---

## Team Project
This game was developed collaboratively by a team of three students. The project emphasized software design, teamwork, and building a flexible system that supports both level creation and gameplay execution.

---

## Gameplay
- Create custom levels using a tile-based editor
- Place different tile types to design unique challenges
- Spawn a player into the created level
- Navigate through obstacles to reach the end tile
- Use movement skills and physics-based interactions to complete levels

---

## Tile System Features
The game includes a variety of interactive tile types:

- **Start Tile**: Defines the player’s spawn point in the level
- **End Tile**: Marks the goal location required to complete the level
- **Solid Tiles**: Standard collision tiles that block player movement
- **Breakable Tiles**: Tiles that disappear or break after interaction
- **Velocity Tiles**: Apply a fixed directional force to the player
- **Velocity Modifier Tiles**: Adjust the player’s current speed or movement strength
- **Pull Tiles**: Attract the player toward a specific direction or center point
- **Push Tiles**: Apply force to push the player away in a given direction
- **Timer Tiles**: Trigger time-based events or activate/deactivate effects after a delay
- **Erase Tile Tool**: Allows players to remove placed tiles during level creation
